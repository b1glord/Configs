# Mysql MariaDB Basit Komutlar

## Database Listeleme
```
show databases;
```

## Database Listeleme
### use komutu ile istenilen database de islem yapılabilir
```
use (database_name);
```

### Bu komut ile database deki tablolar listelenir
```
show tables;
```

### Shell Kullanımı
```
mysql -u root -p -e 'show databases;'
```
